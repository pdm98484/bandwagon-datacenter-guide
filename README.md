## 按运营商来：电信/联通/移动分别该看哪个

**电信用户**

CN2 GIA 本来就是电信自己的高端线路，所以这一块搬瓦工做得最扎实。首选 DC6 CN2 GIA-E，晚高峰下载速度实际用下来可以跑到 100Mbps 以上，丢包率极低。DC9 表现也很接近。

排名参考：香港 > 日本东京 > DC6 ≈ DC9 > EUNL_1 > 其他

预算有限就 DC6/DC9，追求极致就香港，中间档位没必要。

**联通用户**

日本软银（JPOS_1）是联通的经典选择，软银和联通之间直连关系很好，延迟通常比 DC6/DC9 还低。EUNL_1 新机房对联通也非常友好。DC6/DC9 在 2025 年接入了联通 CUP（AS9929）之后，表现也上来了，作为备选可以。

排名参考：EUNL_1 ≈ 日本软银 > EUNL_9 > DC6/DC9（CUP）

如果你联通用日本软银效果好，那就在 CN2 GIA-E 套餐里切到 JPOS_1 用就行，不用单独买。

**移动用户**

2025 年搬瓦工在 DC6 和 DC9 接入了 CMIN2（AS58807）精品线路，这是移动用户历史上在搬瓦工里体验最好的配置。晚高峰丢包率极低，延迟大概在 150ms 左右。

排名参考：DC6/DC9（CMIN2）> EUNL_1 > 香港 HKHK_3（CMI 直连）

移动用户直接选 CN2 GIA-E 套餐，指定 DC6 或 DC9，基本上就不会踩坑。

| 机房 | 测速 IP |

| DC6 CN2 GIA-E | 204.152.218.25 |

直接在命令行 `ping <IP>` 就行，看平均延迟在不在你的预期范围内。晚高峰的时候再测一次，结果更有参考价值。
